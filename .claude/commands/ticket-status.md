# チケットステータス表示コマンド

すべてのカテゴリの進捗状況を包括的に表示します。

## 実行手順

### 1. PROGRESS.md ファイルの読み込み
以下のファイルを読み込みます：
- `docs/ticket/bug/PROGRESS.md`
- `docs/ticket/feature/PROGRESS.md`
- `docs/ticket/enhancement/PROGRESS.md`
- `docs/ticket/initial/PROGRESS.md`

### 2. 統計情報の集計
各カテゴリから以下の情報を抽出：
- 総チケット数
- 進行中のチケット数
- 未着手のチケット数
- 完了チケット数

- 平均完了時間
